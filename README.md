# GeoPhotos

Exif GPS metadata editor for macOS, written in Swift, support >= OS X 10.10.

## Downlaod 

Download on [Mac App Store](https://itunes.apple.com/cn/app/geophotos/id1136088196?mt=12)

## Development

```bash
// in terminal
pod install
open GeoPhotos.xcworkspace
```
